启用存档图鉴功能

存档图鉴功能可以帮您收集和查看游戏存档中的各类物品，让您全面了解游戏进程中的收集成果。

功能简介​

存档图鉴功能会扫描游戏存档，识别并分类展示您收集到的所有物品，包括简单物品、复杂物品和 Mod 物品。您还可以对不同物品进行对比，查看详细信息。

前置要求​

ℹ️ NuGet 市场依赖

存档图鉴功能依赖于 NuGet 市场功能。

在使用本功能之前，请先参阅 启用 NuGet 市场 文档，完成 NuGet 市场的启用。

安装依赖并启用​

首先，您需要安装相关依赖并启用存档图鉴功能。在 SuperDucky 的功能列表中找到该选项，点击安装依赖并启用。

启用功能​

依赖安装完成后，在功能列表中找到存档图鉴选项并启用。

重要提示​

⚠️ 请先进入存档游玩

在启用存档图鉴功能后、预览图鉴之前，请务必登录您的游戏存档并游玩约 2 分钟。

这样可以确保：

  • 游戏数据被正确写入存档
  • 相关图片资源被完整加载
  • 图鉴功能能够正常获取和展示内容

如果直接预览而未先进入存档游玩，图鉴内容可能无法正常显示或缺失数据。

图鉴预览​

启用后，您可以查看图鉴界面，这里会展示您收集到的所有物品分类。

物品对比​

图鉴功能支持对不同物品进行对比，帮助您了解物品之间的属性差异。

添加对比物品​

您可以通过以下两种方式将物品加入对比：

  • 右键点击物品，选择"加入对比"选项
  • 左键点击物品进入详情页，在详情页中点击"加入对比"按钮

查看对比​

添加对比物品后，点击列表右上角的对比按钮即可查看对比界面。

  • 每次最多可以加入 6 个物品进行对比
  • 可以选择其中任意一个物品作为基准进行对比，方便直观查看属性差异

物品类型​

存档图鉴将物品分为多种类型，每种类型都有其独特的展示方式：

简单物品​

简单物品通常是基础的游戏道具，展示基本的物品信息。

复杂物品​

复杂物品具有更多的属性和详细信息，图鉴会展示完整的属性列表。

Mod 物品​

如果您安装了游戏 Mod，图鉴也会识别并展示 Mod 添加的物品。

使用提示​

  • 首次启用可能需要较长时间扫描存档
  • 图鉴会自动更新，反映最新的游戏进度
  • 支持导出图鉴数据为多种格式
  • 可以使用图鉴搜索栏快速查找特定物品
    • 支持输入物品 ID 进行搜索
    • 支持输入物品名称进行搜索
